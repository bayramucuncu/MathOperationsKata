MathOperationsKata
==================

TDD Math oprations application  on .Net

This application is a solution of Math operations like that <br>
x+y <br>
2x+3y+5 <br>
x+(y/z) <br>
((x-y)/4)+(x/y) <br>
...<br>
..<br>
.<br>
x, y, z,... is variable reference<br>
2, 3, 5,... is constant reference.<br>
